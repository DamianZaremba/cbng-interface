Signup

We use OAuth with Wikimedia.org to authenticate users.

In order to login please Signup with Wikimedia, this will also grant you access to project sites such as Wikipedia.

You can learn more about OAuth @ https://www.mediawiki.org/wiki/Help:OAuth

Anonymous users

We understand that not all users want to create accounts. To enable this, basic actions, such as reporting a false positive will be allowed without login. More advanced features require an account, to ensure traceability.
